# clean-fe-template

## 2주차 학습계획

### 개발 방향

- 역할 나누기

  - 상태관리 Store
  - UI 렌더링을 위한 Component
  - 그 외 필요에 따라 추가될 수 있음

- 구현방법

  - Store: Proxy 또는 데코레이터
  - Component: ESClass
  - 옵저버 패턴 차용

- 1차적으로는 대략 위와같이 진행해볼 예정이며, 계속 리팩터링하면서 바뀔 수 있음
- Proxy와 데코레이터를 사용해보지 못해서 이번 기회에 사용성을 익혀볼 예정
- 필요 시, 오브젝트 책 참고, ChatQPT, 구글링 열심히 할 예정..ㅎㅎ
